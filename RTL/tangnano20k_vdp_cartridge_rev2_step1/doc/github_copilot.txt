HDMIやVGAなどのタイミングでは、1ラインのクロック数や1フレームのライン数は、
アクティブ期間（表示領域）＋ブランキング期間（非表示領域） の合計です。

1. 水平タイミング
総クロック数（1ライン）：1368クロック
アクティブ期間：800クロック
ブランキング期間
1368 - 800 = 568クロック

2. 垂直タイミング
総ライン数（1フレーム）：525ライン
アクティブ期間：480ライン
ブランキング期間
525 - 480 = 45ライン

3. フロントポーチ・バックポーチ・同期期間
ブランキング期間は、

フロントポーチ（Front Porch）
同期期間（Sync Pulse）
バックポーチ（Back Porch）
に分かれます。
これらの値はVESAやHDMIの標準値がよく使われますが、
ご提示の条件だけでは分配比率は決まっていません。

参考：VESA 800x480@60Hz（近い値）
水平：フロントポーチ 40、同期 128、バックポーチ 88
垂直：フロントポーチ 1、同期 3、バックポーチ 41
合計

水平ブランキング：40 + 128 + 88 = 256（今回の568より少ない）
垂直ブランキング：1 + 3 + 41 = 45（今回と一致）
